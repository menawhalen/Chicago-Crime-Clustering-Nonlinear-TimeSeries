This repo is a collection for the research "Uneven Progress: Shifting Patterns in Crime Post COVID-19 through Clustering Panels of Nonlinear Time Series" by Mena Whalen, Andrew Papachristos, and Noelle Samia 

The repo includes

- R scripts
- R data files (.rds) in the data folder

The scripts include
- qc_plots: Creates the figures and tables for the paper
- cleaning_data: Prepares the index and firearm related data for analysis. The index crime data can be publically found through the City of Chicago Data Portal. The firearm related crimes from an exisiting agreement and are not publicly available at this time.
- clustering_evaluations_index: Calculates and analyzes the different distance measures and clustering techniques for comparison to be evaluated for a final model for index crimes.
- clustering_evaluations_firearm: Similar analysis but for firearm crimes
